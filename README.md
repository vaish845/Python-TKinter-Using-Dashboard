# 📊 Exploratory Data Analysis (EDA) Dashboard

A Python-based interactive dashboard for Exploratory Data Analysis and Visualization, built with **Pandas**, **Matplotlib**, and **Tkinter**. Designed to make data analysis simple, visual, and accessible — even for beginners.

---

## 🚀 Features

- 📂 Load and analyze datasets with ease
- 📊 Generate visualizations including Bar Charts and distribution plots
- 🧮 Perform basic Exploratory Data Analysis (EDA)
- 🖥️ Interactive Tkinter GUI interface
- 📈 Data visualization powered by Matplotlib
- 📓 Jupyter Notebook support for in-depth analysis

---

## 🛠️ Technologies Used

| Tool | Purpose |

| Python | Core programming language |

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/EDA-Dashboard.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd EDA-Dashboard
```

### 3️⃣ Install Required Libraries

```bash
pip install pandas matplotlib
```

---

## ▶️ Run the Project

**Run the Tkinter GUI application:**

```bash
python tk.py
```

**Or open the Jupyter Notebook:**

```bash
jupyter notebook tk.ipynb
```
